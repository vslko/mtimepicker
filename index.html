Features
  • Wrap on standart input
  • Time selection without keyboard use
  • Easy styling
  • Compact view on page
Getting started
Insert in head
<!-- basic style -->
<link rel="stylesheet" type="text/css" href="your_path/css/mtimepicker.css" />
<!-- required javascripts -->
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.8.1/jquery.min.js"></script>
<script type="text/javascript" src="your_path/js/mtimepicker.js"></script>
Insert in body
<input id="my_time" type="text" readonly="readonly" />

<script type="text/javascript">
$(document).ready( function(){
	$('#my_time').mTimePicker().mTimePicker( 'setTime', '15:40' )
});
</script>
